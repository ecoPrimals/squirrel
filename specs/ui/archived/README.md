# Archived UI Documentation

## Overview

This directory contains archived documentation for the Squirrel UI system. These documents are preserved for historical reference but are no longer actively maintained or updated. Current and active documentation can be found in the parent directory and its subdirectories.

## Contents

This directory contains the following categories of archived documents:

1. **Architecture Documents**: Previous architectural plans and specifications
2. **Implementation Plans**: Original implementation plans that have been superseded
3. **Migration Plans**: Historical migration plans that have been completed
4. **Testing Strategies**: Previous testing approaches that have been updated
5. **Status Updates**: Historical status reports

## File Naming Convention

All archived files follow the `ARCHIVED_` prefix naming convention to clearly identify them as archived content. This helps prevent confusion with current documentation.

## Usage Guidelines

When referencing archived documentation:

1. **Historical Context**: Use these documents for understanding historical context and decisions
2. **Not Authoritative**: Do not treat archived documents as authoritative sources for current implementation
3. **Reference Only**: Cite these documents only as references, not as current specifications
4. **Prefer Current Docs**: Always prefer current documentation when available
