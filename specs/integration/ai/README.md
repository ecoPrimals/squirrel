# AI Integration Specifications

## Overview

This directory contains specifications for integrations between AI components and other parts of the Squirrel platform. These specifications detail how AI capabilities are integrated, how AI agents interact with the system, and how AI tools are exposed through the Machine Context Protocol (MCP).

AI integrations are at various stages of implementation:
