# Test Plugin Fix

A simple test binary for debugging and fixing plugin loading issues in the Squirrel project.

## Purpose

This binary serves as a test harness for:

- Testing plugin loading mechanism
- Debugging plugin integration issues
- Verifying plugin API changes
- Testing context adapter plugin functionality

## Usage

```bash
cargo run -p test_plugin_fix
```

Currently, this is a placeholder that will be expanded with actual plugin testing functionality as needed.

## Related Components

- `squirrel-plugins`: Core plugin infrastructure
- `squirrel-context-adapter`: Context adapter with plugin support
