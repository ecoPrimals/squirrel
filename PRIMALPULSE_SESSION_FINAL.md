## 🚀 Usage

```bash
# Start Squirrel
target/release/squirrel

# List tools
curl http://localhost:9010/api/v1/actions

# Analyze primal code
curl -X POST http://localhost:9010/ai/execute \
  -H "Content-Type: application/json" \
  -d '{
    "action": "primal.analyze",
    "input": {
      "primal_path": "/path/to/primal",
      "depth": "summary"
    }
  }'

# Audit hardcoding
curl -X POST http://localhost:9010/ai/execute \
  -H "Content-Type: application/json" \
  -d '{
    "action": "primal.audit_hardcoding",
    "input": {
      "primal_path": "/path/to/primal",
      "scope": "all"
    }
  }'

# Generate semantic commit
curl -X POST http://localhost:9010/ai/execute \
  -H "Content-Type: application/json" \
  -d '{
    "action": "rootpulse.semantic_commit",
    "input": {
      "diff": "Added new features...",
      "primal_name": "squirrel",
      "style": "conventional"
    }
  }'
```
