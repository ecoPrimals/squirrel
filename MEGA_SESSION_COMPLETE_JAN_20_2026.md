**Status**: ✅ **CAPABILITY DISCOVERY IMPLEMENTED**

**New Module**: `crates/main/src/capabilities/`

```rust
// Discover capability providers at runtime (zero hardcoding!)
use squirrel::capabilities::discover_capability;

// Find who provides crypto signing (could be anyone!)
let crypto = discover_capability("crypto.signing").await?;

// We have NO IDEA what primal this is - we only know it can sign
let signature = crypto.call("sign", data).await?;
```

**Discovery Methods**:
1. **Environment variables**: `CRYPTO_SIGNING_PROVIDER_SOCKET`
2. **Socket scanning**: Scans `/tmp`, `/var/run`, `$XDG_RUNTIME_DIR`
3. **Registry queries**: Queries capability registry if available
4. **Socket probing**: Connects and asks "what can you do?"

**Features**:
- ✅ Zero hardcoded primal names
- ✅ Zero vendor dependencies
- ✅ Runtime discovery only
- ✅ 500ms timeout per probe (non-blocking)
- ✅ JSON-RPC capability negotiation
- ✅ Multi-source discovery
